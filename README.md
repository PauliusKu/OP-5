## Programos įdiegimas
  1. Atsisiųskite [**naujausią versiją.**](https://github.com/PauliusKu/2-uzduotisVEC/releases)<br>
  2. Išskleiskite failus į pasirinktą vietą
  3. Per konsolė įveskite:
  
    cd OOP-Task-5/Map_Project/cpp
    g++ -std=c++11 *.cpp -o main
    ./main
